Discord Linux Repository

Automatically updated apt repository for Linux distributions

Installation

  1. Add the repository key:
    $ sudo apt-key adv --keyserver keyserver.debian.com --recv-keys XXXXXXXXXXXXXXXX
  2. Add the repository source:
    $ echo "deb https://example.com/discord-repo stable main" | sudo tee /etc/apt/sources.list.d/discord.list
  3. Update the package list:
    $ sudo apt update
  4. Install Discord:
    $ sudo apt install discord
